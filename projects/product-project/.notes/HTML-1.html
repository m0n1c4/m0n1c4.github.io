Intro to HTML

What is HTML

HyperText Markup Language is a language used to structure and give meaning to content by wrapping said content in elements that describe the content's function and purpose.

HTML is interpreted by web browsers that then dictate how different pieces of content should appear and react to user interaction in the DOM.

Elements

HTML elements encapsulate and define content. There are various types of HTML elements with many distinct purposes, but they, for the most part, use a very similar structure.

Tags

HTML tags denote the beginning(and of often, at the end) of elements. Tags consist of opening and closing angle brackets around the name of the element type:

1
<img>

Opening tags can also contain attributes that describe the contents of the tag, but when an element requires a closing tag, the closing tag is always a pair of angle brackets containing the tag name preceded by a forward slash:

1
<header></header>

Attributes

While tags give structure to HTML documents, attributes are used to provide additional information about elements. Attributes begin with the attribute name, often followed by an equals sign and a value in double quotes:

1
<img src="cats.gif">

Some attributes, such as disabled, do not require a value:

1
<button disabled></button>

Anchors

Anchor elements, colloquially known as "links", are used to navigate between pages. These elements require a href attribute to be valid HTML:

1
<a href="http://http://www.w3.org/People/Berners-Lee/">Tim</a>

Images

Image elements place images into the DOM. Similar to the anchor element, images require a src attribute to be valid. Also, these elements are self-closing as their content is dictated by the src attribute:

1
<img src="http://www.fillmurray.com/600/600">

Lists

There are two types of lists in HTML: ordered(<ol>) and unordered(<ul>). An ordered list is used when the items in the list require the order they are written in to be preserved, otherwise an unordered list is used.

Both types of list contain list items: <li>.

Sections

There are several distinct elements used to break up content into sections in HTML documents. Some of the most common elements used for this purpose include <section>, <article>, <header>, <footer>, and <nav>.

Documents

Filetypes

HTML documents begin with a filetype declaration. There used to be several types of HTML files, but in modern web development, html is the only HTML doctype used.

The HTML Element

Filetype declarations are followed by an HTML element that contains the rest of the elements in the HTML file:

1
2
3
4
<!DOCTYPE html>
<html>

</html>

The Head

The head element contains meta data related to the webpage such as the title to be displayed in the browser window header, the character set used, information about the author, and files that relate to the webpage:

1
2
3
4
5
6
7
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>My First Webpage</title>
  </head>
</html>

  </head>

    <header>
      <h1>My First Webpage</h1>
    </header>
  </body>

An HTML file with a file type declaration, HTML element, head, and body, is a complete and valid HTML file.
