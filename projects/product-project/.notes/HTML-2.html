Interactive Web Pages Using Forms

Input

One of the most versatile elements in HTML, the <input /> element, allows us to specify different kinds of input the user can submit to the application. The attribute that specifies the type of input is type <input type="text" />.

The type attribute tells the input element the type interaction it will provide. You can give the users the ability to enter free form text, a password, email or even pick out of a list of choices.

Text Fields

Helps us enter arbitrary text like names, last names, id's and even phone numbers.
